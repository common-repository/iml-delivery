==Description==

Плагин добавляет 4 метода доставки (Курьерская доставка IML с кассовым
обслуживанием, Предоплаченная курьерская доставка IML, Доставки IML до ПВЗ с
кассовым обслуживанием, Предоплаченная доставка IML до ПВЗ) в WooCommerce для
расчета стоимости доставки на страницах корзины (cart) и заказа(checkout).
Если у созданного заказа в WooCommerce выбран один из 4 методов доставки, то
данный заказ можно отправить в IML API и контролировать его доставку.
Плагин позволяет проверять статусы заказов в IML API.
Плагин позволяет получить этикетки со штрих-кодами для грузовых мест.

== Installation ==

1. скачать модуль интеграции для Wordpress
1. Распаковываем файл-архив и копируем полученную папку в следующую директорию - КОРНЕВАЯ ПАПКА ВАШЕГО WORDPRESS-САЙТА/wp-content/plugins
1. Заходим в админку вашего сайта на страницу управления плагинами. В списке плагинов должен быть неактивный плагин
1. Активировать данный плагин, нажав на ссылку с надписью Activate (или "Активировать").

Подробную инструкцию по настройке модуля Вы можете найти по данной [ссылке](https://ipol.ru/spravka/iml_wordpress/module_about/)

В случае проблем с модулем обратитесь по адресу [cms@iml.ru](mailto:cms@iml.ru), описав суть проблемы.
